# Linear-Regression-Single-Variable
# 🏠 House Price Prediction (Simple Linear Regression)

This repository features a machine learning model designed to predict real estate prices based on property size (Square Footage). It serves as a foundational example of **Supervised Learning** using the Simple Linear Regression algorithm.

---

## 🛠️ The Machine Learning Stack
* **Python**: Core programming.
* **Pandas & NumPy**: For data loading and numerical operations.
* **Matplotlib & Seaborn**: For visualizing the regression line and data distribution.
* **Scikit-Learn**: For implementing the `LinearRegression` model and performance metrics.

---

## 📊 Project Workflow

### 1. Data Visualization
* Created **Scatter Plots** to check for a linear relationship between Area (sq ft) and Price.
* Identified trends to ensure the data is suitable for a linear model.

### 2. Model Implementation
* **Splitting Data:** Divided the dataset into Training and Testing sets (e.g., 80/20 split).
* **Training:** Fitted the Linear Regression model to the training data to find the "Line of Best Fit."
* **Equation:** The model calculates the relationship using the formula:  
  $$y = mx + b$$  
  *(Where $y$ is Price, $x$ is Area, $m$ is the coefficient, and $b$ is the intercept).*

### 3. Evaluation & Results
* **Regression Line:** Visualized the predicted values against the actual data points.
* **Metrics:** Used **Mean Squared Error (MSE)** and **R-squared ($R^2$)** to determine the accuracy and reliability of the predictions.
